## 📝 JSON Format Examples

### Super Simple Format:

**Training Campaign:**
```json
{
  "campaign": "Campaign 6",
  "year": 2025,
  "period": "Dec 2025",
  "participants": 420,
  "completionRate": 95,
  "status": "completed"
}
```

**Restaurant Visit:**
```json
{
  "month": "Apr 2025",
  "visits": 75,
  "avgScore": 89.5,
  "highest": 99,
  "lowest": 83
}
```

**Weekly Wrap:**
```json
{
  "week": "Week 11",
```
